A lightweight Elementor testimonial slider widget with repeater, ratings, arrows, dots and full styling controls.

== Description ==

Testimonial Slider for Elementor adds a fully customizable testimonial slider to Elementor. Add unlimited testimonials using a repeater, each with a photo, name, job title, star rating and review text.

**Features:**
* Repeater — add unlimited testimonials
* Star ratings per slide (1–5)
* Autoplay with configurable speed
* Infinite loop
* Navigation arrows (inline SVG, no Font Awesome dependency)
* Pagination dots
* Alignment control (left / center / right)
* Full typography controls
* Border, padding, border radius, box shadow controls
* Color controls for text, stars, dots, arrows
* Live preview in Elementor editor

**Requires:** Elementor (free) 3.0 or higher.

== Installation ==

1. Upload the `elementor-testimonial-widget` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Open any page with Elementor.
4. Search for **"Testimonial Slider"** in the widget panel and drag it onto the canvas.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =
No. It works with the free version of Elementor (3.0+).

= Can I add more than 3 testimonials? =
Yes. Use the Repeater in the Content tab to add as many slides as you need.

= The arrows are not showing on the frontend =
This widget uses inline SVG arrows so Font Awesome is not required. If arrows are still not showing, check that "Show Arrows" is enabled in the Slider Settings section.
