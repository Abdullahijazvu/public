== Description ==

Prime Elementor Addons is a lightweight **Elementor addon plugin** for WordPress that gives you a focused set of **Elementor widgets** to build modern pages faster. Instead of installing many separate Elementor extensions, you can design key sections using one clean widget pack.

It includes the widgets most websites actually need - like **Advanced Accordion**, **Advanced Tabs**, **Advanced Heading**, **Call to Action**, **Pricing Table**, **Testimonial**, **Team Member**, **Countdown**, **Counter Number**, **Progressbar**, **Post Grid**, **Social Icons**, and more - so you can create complete landing pages and business pages with drag-and-drop.

Everything works inside the Elementor editor with simple controls. Add a widget, write your content, style it, and publish - no extra setup, no complicated steps, and no coding required.

If you’re looking for **addons for Elementor** that are easy to use and built for real layouts, **Prime Elementor Addons** is a smart choice. With its powerful **Full Site Builder**, you can design headers, footers, and templates without needing a Pro theme. It helps you create clean sections, keep your design consistent, and build WordPress sites quicker.

== Installation ==

It's very easy and straight forward to install this plugin. But before installing **Prime Elementor Addons** make sure you've installed and activated [Elementor](https://wordpress.org/plugins/elementor/ "Install Elementor").

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Prime Elementor Addons`.
3.  Install and activate the plugin, that's it!

= Manual Installation =
1.  Download [Prime Elementor Addons](https://downloads.wordpress.org/plugin/unlimited-elementor-inner-sections-by-boomdevs.zip "Prime Elementor Addons").
2.  Extract the zip file. You'll get plugin files inside directory.
3.  Upload the plugin files to the `/wp-content/plugins/unlimited-elementor-inner-sections-by-boomdevs` directory.
4.  Activate the plugin through the 'Plugins' screen form your WordPress website dashboard.
